# Security Policy

This is the organisation-wide default. A repository that ships its own `SECURITY.md` overrides
this file — see
[playwright-ai-triage/SECURITY.md](https://github.com/flaketrace/playwright-ai-triage/blob/main/SECURITY.md)
for that project's specific scope, which is the one you probably want.

## Supported versions

Projects here are pre-1.0. Only the latest published release of any package receives security
fixes; there are no maintenance branches.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting: **Security tab → Report a vulnerability** on the
affected repository. If that option is unavailable, email **y.krivushenko@gmail.com** with the
repository name in the subject.

**Do not open a public issue for anything you believe is exploitable.**

Expect an acknowledgment within a few days. Please include:

- the affected repository and version,
- a minimal reproduction,
- the impact you see.

Please give a reasonable window to ship a fix before disclosing publicly. Reports are welcome from
anyone; there is no bounty programme.
